All devices within your topology need to be added to the management database to allow connections and configurations

  2. Click the Add New Device button
  3. Enter the device information in the dropdown
  4. Click Add Device button when finished

Device connected:
  1. Navigate to the device manager page
  2. Click the configuration button on a device
  3. You will be redirected to that devices configuration page
Device not connected:

  1. If this is symbol is displayed, it means NetManager has been unable to establish a connection to this device


Configuring:
  1. Navigate to the device configuration page
  2. From the 'Device Interfaces' widget, select the interface you wish to configure
  3. In the pop-up, specify the interfaces IP address, subnet mask and enable status
  4. Click 'Yes' in the pop-up to confirm
Resetting:
  1. Navigate to the device configuration page
  2. From the 'Device Interfaces' widget, select the interface you wish to reset
  3. Click 'Yes' in the pop-up to confirm the reset

  2. In the device configuration page, navigate to the 'Device Security Features' widget

Creating Access Lists
  1. Fill out the 'Edit and Create Access List' form (shown below)

  2. Click the 'Push Changes' button to create the new access list
Editing Access Lists
  1. Use the 'Edit and Create Access List' form using the name of the access list you want to edit
  2. Click the 'Push Changes' button to add the new line
Deleting Access Lists
  1. Select the access list you wish to delete from the 'Delete Access List' form
  2. Click the 'Delete Access List' button
